This is a Folder meant to keep all the update scripts to update the database:
Name scrips as follows:

->	A number pertaining to the order the scripts must be run: 
	if 4 scripts already exist then name this field "5"
->	The date of publishing the scipt: in following format -> 01Jan2019
->	Initials or script creator
->	Some useful info about the script: eg. Add_UserTable

Example:
	1_27Aug2019_ME_CreateDatabase
	2-29Aug2019_LH_AddAccountsTable
